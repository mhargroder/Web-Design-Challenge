Project Summary


This project analyzed the 4 different weather conditions realitive to latitude. Measurements for temperature, humidity, cloudiness and and wind speed were recorded for randonmly generated geo coordinates. The results were then plotted realitive to latitude.
First coordinates were randomly generated using a numpy random function to create as list tuples. The list of tuples were then evaluated by the CityPy library to return coordinates of the nearest city. For each unique city identified an API call was made to the OpenWeather endpoint to fetch a response containing the afore mentioned measures and appeneded to a Pandas data frame. Finally Matplotlib was used to generate scatter plots for each measure in an effort to see if there were any obvious relationships between latitude and the individual weather measurements.
